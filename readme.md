# Movie Search App

A movie search app built with HTML, CSS, and JavaScript. It uses the OMDb API to search for movies by title and display details like poster, year, rating, genre, runtime, and plot.

## Features

- Search movies by name
- Display movie poster
- Show year, rating, genre, runtime, and plot
- Loading, error, and empty states
- Clean and simple interface

## Built With

- HTML
- CSS
- JavaScript
- OMDb API

## How It Works

Type a movie name in the search box and click the search button.  
The app sends a request to the OMDb API and fetches the movie data.  
If the movie is found, the details are shown on the screen.  
If no movie is found, the app shows an empty state message.
